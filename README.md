# Java OOP Concepts

This repository contains my Java Object-Oriented Programming coursework.

## Lab Tasks
* **Lab Task 1:** Basic classes and methods. Contains a command-line calculator.
* **Lab Task 2:** Constructor overloading. Contains a student registration system.
* **Lab Task 3:** Encapsulation. Contains programs for shapes, bank accounts, and student marks using getters and setters.
* **Lab Task 4:** Inheritance. Calculates profit for a savings account.

## Exams and Major Projects
* **Open Ended Lab (OEL):** Inheritance and static variables. A Library Management System managing books, magazines, and late fees.
* **Final Exam Question 2:** Abstract classes and method overriding. A Hospital Management System calculating salaries and bonuses for doctors and surgeons.

## Practice Scenarios
* **Practice Question 7:** Generic items and inheritance for an inventory system.
* **Practice Question 8:** Aggregation, composition, and interfaces for an advanced Library system.
* **Practice Question 9:** Custom exceptions and nested classes for a Car Rental system.
* **Practice Question 10:** Polymorphism and interfaces for calculating retail sales discounts.
